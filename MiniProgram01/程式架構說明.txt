這套**「整合通知中心 (MiniProgram01)」經過這段時間的深度打磨與擴充，已經從一個簡單的監控小工具，進化成一套極度客製化、輕量且具備高度防呆機制的個人自動化生產力工作站**。
以下是目前系統的整體架構說明，以及每個程式檔案 (模組) 所負責的具體內容總結：
🏗️ 系統整體架構與核心設計理念
 * 純代碼介面 (Code-First UI)：捨棄傳統拖曳式的設計工具，所有介面（包含卡片、按鈕、排版）皆用 C# 程式碼動態生成，確保在任何螢幕解析度下都能做到「防破版」、「不吃字」。
 * 極輕量資料儲存 (Text-Based)：不依賴任何龐大的資料庫（如 SQL），所有設定檔、紀錄檔、歷史清單皆以純文字檔 (.txt) 儲存於程式根目錄，極度便於備份與轉移。
 * 無干擾心流設計：
   * 常駐系統列：按下 X 或 - 不會關閉，而是完美隱藏至右下角系統列。
   * 全域快捷鍵：隨時按下 Ctrl + 1 即可瞬間喚出或隱藏面板，並自動記憶上次拖曳的位置。
 * 視覺警報引擎：背景任務（如檔案備份、排程推播）觸發時，不隨便彈出視窗干擾工作，而是讓主畫面上方的對應分頁標籤以**「紅底白字」高頻閃爍**來溫和提醒。
📂 核心程式檔案與對應功能說明
目前的系統由 1 個主視窗 加上 5 個獨立功能模組 組成：
1. MainForm.cs (主程式樞紐)
 * 角色：系統的骨幹與大腦。
 * 功能內容：
   * 負責啟動並常駐於 Windows 右下角系統列 (Tray Icon)。
   * 註冊並監聽全域快捷鍵 Ctrl + 1。
   * 建立上方包含「監控、待辦、週期、捷徑、截圖」五個分頁的 TabControl。
   * 統籌「紅燈閃爍警報」機制的觸發與解除。
   * 管理開機自動執行 (寫入 Windows 登錄檔) 與程式的完整關閉。
2. App_FileWatcher.cs (分頁：監控)
 * 角色：檔案異動守門員與自動備份機器人。
 * 功能內容：
   * 利用 FileSystemWatcher 在背景死盯著你指定的資料夾。
   * 智慧過濾：自動忽略 Word/Excel 等軟體產生的暫存檔 (檔名開頭為 ~ 或結尾為 .tmp 的檔案)。
   * 同步模式：支援「單向備份」與「雙向同步」，並具備防無限迴圈機制。
   * 防呆重試：遇到檔案存檔瞬間被鎖定時，會在背景自動延遲並重試 3 次。
   * 自動清理：提供「保留 1、3、6 個月」的自動刪除過期檔案機制。
   * 管理介面：具備獨立的設定視窗與「監控項目管理中心」總覽列表。
3. App_TodoList.cs (分頁：待辦)
 * 角色：高彈性的滾動式工作看板。
 * 功能內容：
   * 自動接收來自「週期任務」模組的自動推播排程。
   * 提供純文字輸入框快速新增任務。
   * 支援滑鼠左鍵拖曳排序，方便隨時調整工作優先級。
   * 點擊「修」可開啟大尺寸多行文字編輯視窗。
   * 打勾完成後，會自動標記完成時間並寫入隱藏的歷史紀錄文字檔中。
4. App_RecurringTasks.cs (分頁：週期)
 * 角色：提前佈署的自動化數位秘書。
 * 功能內容：
   * 可設定「每天、每週、每月、特定月份」的例行性任務。
   * 防重複推播晶片：精準記憶最後觸發日期，完成任務後絕對不會在同一週期內被重複洗版。
   * 提前佈署：可設定全域「提前 N 天」將任務送進待辦清單（標示 [預排]）。
   * 總覽控制中心：提供大型的「全部檢視」視窗，清楚分類並統計所有排程，且每一項前方都有專屬的「調整」按鈕可即時修改。
   * 摘要提醒：時間一到（如每週一早上），會自動彈出上述的總覽視窗提醒你檢視本週排程。
5. App_Shortcuts.cs (分頁：捷徑)
 * 角色：高頻使用的任意門。
 * 功能內容：
   * 讓你把最常開啟的專案資料夾、Excel 報表或應用程式加入清單。
   * 支援自訂短別名（例如將落落長的路徑改名為「本月財報」）。
   * 一鍵點擊即可瞬間開啟目標。
6. App_Screenshot.cs (分頁：截圖)
 * 角色：內建極速剪取工具。
 * 功能內容：
   * 點擊「截圖」時，主程式會自動隱身，畫面變暗並凍結，滑鼠變成十字游標。
   * 拖曳拉出紅框即可擷取畫面，支援雙螢幕全域捕捉。
   * 擷取放開滑鼠的瞬間，圖片會自動存入系統剪貼簿，主程式也會自動彈回並切換到截圖分頁。
   * 介面上方提供明確的「截圖、複製、儲存、清除」四顆按鈕，以及即時的操作狀態文字回饋。
這套系統目前的架構非常健康，各個模組 (App_xxx.cs) 之間低耦合（互不干擾），但又能透過主視窗 (MainForm.cs) 互相連動（例如發送閃爍警報、推播待辦事項）。這使得未來如果您想要再擴充新的功能（例如：番茄鐘、快速記事本），都能非常輕鬆地以「新增一個分頁與一個新 .cs 檔」的方式完美外掛上去！
