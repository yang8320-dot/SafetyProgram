這是一份為你量身打造的**「整合通知中心 (MiniProgram01)」系統架構與功能總結。經過我們這幾次的迭代與深度優化，這套軟體已經從一個簡單的小工具，進化成了具備商業軟體水準的生產力與自動化工作站**。
以下是目前的系統架構與功能全貌：
🏗️ 系統底層架構 (Core Architecture)
 * 介面框架：基於 Windows Forms 開發，並在 Program.cs 中強制啟用 High DPI Aware (防模糊機制)，確保在高解析度螢幕上文字清晰。
 * 全域快捷喚醒：註冊了全域快捷鍵 Ctrl + 1。不論你在操作什麼軟體，按下快捷鍵即可秒速喚醒程式，且維持在你上次使用的分頁，不會強制跳轉。
 * 常駐系統列：啟動後自動縮小至右下角系統列 (System Tray)，支援「開機自動執行」設定。
 * 模組化設計：採用 TabControl 將功能分為四大模組，每個模組都是獨立的 UserControl 檔案，便於日後維護與擴充。
 * 輕量級持久化儲存：完全不依賴資料庫，所有設定與紀錄皆以 .txt 純文字檔儲存在程式根目錄，輕量且方便備份轉移。
🚀 四大核心模組與功能
1. 📁 檔案監控備份模組 (App_FileWatcher.cs)
功能定位：防呆備份與異動追蹤的隱形守衛。
 * 極簡主畫面：只顯示「異動紀錄卡片」與「一鍵清除」按鈕，保持視覺乾淨。
 * 彈出式設定中心：點擊設定後跳出獨立居中視窗，徹底解決了高 DPI 縮放造成的「吃字」與排版錯亂問題。
 * 智慧路徑處理：支援資料夾選取，並具備「一鍵貼上並自動去除引號」的功能。
 * 高階過濾引擎：
   * 自訂頻率：可手動輸入秒數，防止檔案頻繁寫入衝突。
   * 自訂深度：獨家開發的「路徑深度計算引擎」，支援無限層、僅本層，甚至精確到「第十層」的自訂深度攔截。
 * 結構化還原：備份時會自動在備份區**「1:1 還原來源端的子資料夾結構」**。
 * 大尺寸管理清冊：800x850 的獨立管理視窗，長路徑也能以藍綠雙色清晰換行顯示，並可一鍵刪除任務。
2. 📝 待辦清單模組 (App_TodoList.cs)
功能定位：靈活、支援長篇文字的滾動式看板。
 * 容器化排版 (FlowLayoutPanel)：捨棄傳統死板的清單，將每個任務變成獨立區塊。
 * 長文字自動換行：輸入再長的任務細節都會自動往下長，文字絕對不會被截斷。
 * 物理拖曳排序：滑鼠按住即可上下拖曳調整優先級，拖曳時帶有**「藍色精準插入指示線」**，且程式會自動記憶你的排序。
 * 大容量編輯視窗：點擊右側專屬的「修」按鈕（或雙擊文字），會彈出支援多行、帶有捲軸的 450x250 大型編輯框，游標自動定位至句尾，方便滾動式修正。
 * 歷程記錄：完成或新增的任務會自動寫入 todo_history_added.txt 與 todo_history_completed.txt 留存。
3. 🔁 週期任務模組 (App_RecurringTasks.cs)
功能定位：你的自動化秘書，預排未來的例行公事。
 * 彈出式設定中心：比照監控模組，設定介面改為獨立彈出視窗，主畫面僅保留乾淨的任務列表。
 * 豐富的排程選項：支援「每個月/每週」搭配「每天/特定號數/星期/工作日/週末」，以及自訂精確的觸發時間 (HH:mm)。
 * 預排與提前新增 (亮點功能)：
   * 可設定**「提前 N 天新增」**（例如提前 3 天或 7 天）。
   * 時間一到，程式會自動將該週期任務塞進你的「📝 待辦模組」中，並加上 [預排-MM/dd] 的標籤提醒你提早準備。
 * 清單摘要提醒：可設定在每週一或每月 1 號的指定時間，發送「共有 X 項排程」的摘要至待辦清單中。
4. 🚀 捷徑管理模組 (App_Shortcuts.cs)
功能定位：高頻使用的檔案與資料夾任意門。
 * 支援手動選取或貼上路徑。
 * 可為路徑自訂易讀的別名（點擊卡片名稱即可修改）。
 * 一鍵呼叫 Windows 檔案總管開啟目標目錄或執行檔案。
總結
目前的程式碼狀態非常穩定，排版問題（特別是縮放吃字）已透過**「彈出式獨立視窗」與「GroupBox 容器保護」徹底解決。待辦清單的「拖曳排序」與週期任務的「提前佈署」**更是大幅提升了軟體的實用性。
這套程式已經是一個非常完整的 MVP (最小可行性產品)，隨時可以投入日常工作中使用！
